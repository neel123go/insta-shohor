How does JavaScript work?

JavaScript is a client-side scripting language. Which we run in the browser. Every browser has a JS Engine that runs the JavaScript code in the browser. Example: chrome use V8 Engine. Every JavaScript engine has a parser to parse our JavaScript code. The parser will create an Abstract Syntax Tree. Then our code will be translated into meshing code. Then the meshing code will run.

What is the difference between Local Storage & Session Storage?

Web Storage provides two objects for storing data on the client: 1. Local Storage, 2. Session Storage. There are some differences between Local Storage and Session Storage. Local Storage contains data permanently. On the other hand, Session Storage contains data tempararity. The Storage capacity of Local Storage is 5MB/10MB. On the other hand, the storage capacity of Session Storage is 5MB. Local Storage is not session-based, it must be daleted via JavaScript or manually. On the other hand, Session Storage is session-based. This means that data is stored until the browser (or tab) is closed.
